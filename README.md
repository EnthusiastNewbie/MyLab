Questa repo è il mio **banco di lavoro**. Qui raccolgo script utili, dotfiles e tool che sviluppo mentre imparo a gestire Linux e l'ecosistema open source.

Se trovi qualcosa di utile, copialo pure. Se trovi un bug, beh... è un laboratorio! 💥
