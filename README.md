```

E inserisci:

```

Esempi di frequenza:
- Ogni 30 minuti ‚Üí `*/30 * * * *`
- Ogni giorno alle 8 ‚Üí `0 8 * * *`

Usa il crontab di **root** se vuoi leggere SMART.
